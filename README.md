# [WIP] vim-gobuild

`:GoBuild` command for Go inspired by [mattn/vim-gorun](https://github.com/mattn/vim-gorun)

## Usage

```
:GoBuild
```

## Installation

For [vim-plug](https://github.com/junegunn/vim-plug) plugin manager:

```viml
Plug 'NasSilverBullet/vim-gobuild'
```
